# vue-simple-table

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

### Configurations
Configuration object. More configurations will be add based on demand.
```
config: {
  filename: ''
}
```

Example of component
```
<simple-table
  :inputdata="testingData"
  :configdata="config"
/>
```
